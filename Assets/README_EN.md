# Level & Gem Setup

## Creating Gems

1. Navigate to `Assets/Data/Gems`
2. Right-click → `Create > GridBased_Puzzle > Gem Definition`
3. Configure the gem properties in the Inspector

## Creating Levels

1. Navigate to `Assets/Data/Levels`
2. Right-click → `Create > GridBased_Puzzle > Level Configuration`
3. Add gems to the **Gem Definitions** list by dragging them from `Assets/Data/Gems`

## Assigning Levels

1. Open `Assets/Scenes/SampleScene.unity`
2. Select `GameController` in the Hierarchy
3. Drag level configurations into the level management component
